# SL FTTH Power Calculator

A simple web-based tool designed for **fiber splicers and FTTH technicians** to calculate optical signal power after different optical splitters.

This tool helps technicians quickly estimate **output power at each splitter port**, making FTTH network planning and troubleshooting easier.

---

## 🚀 Features

* Calculate **optical power after splitters**
* Supports multiple splitter types:

  * 1×2
  * 1×4
  * 1×8
  * 1×16
  * 1×32
* Supports **uneven splitters** (example: 80/20, 90/10, 99/1)
* Shows **output power for each port**
* **Responsive mobile-friendly UI**
* Built with **Tailwind CSS**

---

## 🛠 Technologies Used

* HTML5
* Tailwind CSS
* JavaScript
